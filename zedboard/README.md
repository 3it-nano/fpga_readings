# The FPGA Chip

- Dual-core ARM Cortex-A9 MPCore™ with CoreSight™
- 667 MHz (-1); 766 MHz (-2); 866 MHz (-3)
- Artix-7 FPGA
    - LUT:  46,200
    - Flip-Flops:  106,400
    - BRAM: 4.9 Mb
